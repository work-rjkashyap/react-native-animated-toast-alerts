## Installation

### 1. Install the package

Choose your preferred package manager:

```bash npm
npm install react-native-animated-toast-alerts
```

```bash yarn
yarn add react-native-animated-toast-alerts
```

```bash pnpm
pnpm add react-native-animated-toast-alerts
```

### 2. Install peer dependencies

The library requires `react-native-svg` and `lucide-react-native` for icons:

```bash npm
npm install react-native-svg lucide-react-native
```

```bash yarn
yarn add react-native-svg lucide-react-native
```

```bash pnpm
pnpm add react-native-svg lucide-react-native
```

### 3. iOS Setup

For iOS, you need to install the CocoaPods dependencies:

```bash
cd ios && pod install && cd ..
```

### 4. Android Setup

No additional setup is required for Android. The library works out of the box!

## Basic Setup

Wrap your application with the `ToastProvider`:

```tsx
import { ToastProvider } from 'react-native-animated-toast-alerts';

function App() {
  return (
    <ToastProvider>
      <YourApp />
    </ToastProvider>
  );
}

export default App;
```

That's it! You're now ready to start using toast notifications in your app.
