
```bash yarn
yarn add react-native-animated-toast-alerts
```


## Basic Example

```tsx
import { ToastProvider, useToast } from 'react-native-animated-toast-alerts';

function App() {
  return (
    <ToastProvider>
      <YourApp />
    </ToastProvider>
  );
}

function MyComponent() {
  const showToast = useToast();

  const handlePress = () => {
    showToast({
      type: 'success',
      message: 'âœ¨ Operation completed!',
      position: 'top'
    });
  };

  return <Button title="Show Toast" onPress={handlePress} />;
}
```

## Features Overview

### Modern Design
Clean, minimalist interface with smooth animations and beautiful transitions that feel native to your app.

### Animated Stacking
Beautiful Sonner-style entrance and exit animations with proper stacking for multiple toasts.

### Dark Mode Support
Seamless integration with your app's theme, automatically adapting to light and dark modes.

### Gesture Support
Intuitive swipe-to-dismiss functionality with natural physics-based animations.

### TypeScript First
Comprehensive type definitions for a better development experience with full autocomplete support.

### Highly Customizable
Extensive customization options for colors, icons, positions, durations, and animations.
